# README

## 目录含义

- devenv: 开发环境自动部署
- conf: 部分现成配置文件
- shell: 辅助脚本
- spin: 争对不同 Fedora spin 编写的对应脚本（不包含通用配置）

## 默认安装的软件

- vim
- emacs
- vscode
- xmind
- Thunderbird
- obs
- steam
- trilium
- flameshot
- cmake
- git
- bash 自动补全

## 默认卸载的软件

- vi
