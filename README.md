#### Windows

- 用户配置(全局配置): `~\AppData\Roaming\Code\User\settings.json`
- 快捷键配置: `~\AppData\Roaming\Code\User\keybindings.json`
- 自定义代码片段: `~\AppData\Roaming\Code\User\snippets\*.json`
